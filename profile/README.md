# Koina

<p><strong>We build small, practical AI stuff.</strong></p>

<i><p>English | <a href="./README.zh-CN.md">简体中文</a></p></i>

</div>

We like messing around with AI and shipping little things that feel useful, fun, or just worth trying.

Sometimes that means tools we wanted.
Sometimes that means local or self-hosted spins on projects we liked.
Sometimes that means relays, benchmarks, side quests, or weird little experiments that got out of hand.

## 🔧 What we're into

- useful AI tools
- local-first or self-hosted projects
- API relays and infra bits
- small benchmarks and availability checks
- side projects that solve our own annoyances first

## 🧪 What we're building currently

- an AI API price and availability board
- a cleaner relay core we can actually live with
- local versions of projects we want to run ourselves
- small experiments that may or may not turn into real tools

## 🤝 If you want to contribute

Issues, PRs, bug reports, docs fixes, design help, testing, ideas, and sharp opinions are all welcome.

The one thing we really do not want is blind AI-generated code tossed over the wall. If you used AI, cool. Just read it, run it, review it, and make sure you actually stand behind it before opening the PR.
