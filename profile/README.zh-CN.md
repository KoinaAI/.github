我们喜欢捯饬 AI，也喜欢把一些觉得有用、好玩，或者单纯值得一试的小东西做出来。

有时候是我们自己想用的工具。
有时候是我们想自己跑起来的本地版或自托管项目。
有时候是中转、榜单、基准测试、边角料项目，或者一些越做越上头的小实验。

## 🔧 我们喜欢折腾什么

- 实用一点的 AI 工具
- 本地优先或可自托管的项目
- API 中转和一些基础设施小零件
- 小型 benchmark 和可用性检查
- 先解决我们自己烦恼的 side project

## 🧪 我们正在做什么

- AI API 的价格和可用性榜单
- 一个我们自己愿意长期用下去的 relay core
- 一些想自己跑起来的项目本地版
- 一些不一定会长成正式产品的小实验

## 🤝 如果你想一起来

Issue、PR、bug 反馈、文档修修补补、设计建议、测试、点子，甚至是有道理的吐槽，我们都欢迎。

我们唯一不太欢迎的是那种没看、没跑、没审过，就直接扔过来的 AI 生成代码。你用 AI 完全没问题，但至少先自己过一遍，确认你真的愿意为这份改动负责。
